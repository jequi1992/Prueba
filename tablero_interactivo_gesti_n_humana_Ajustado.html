Guía de Conexión: Estructura de Nuevos Campos en Looker Studio

Para consolidar las nuevas 6 variables en tu reporte real de Looker Studio de manera idéntica a este prototipo web interactivo, implementa los siguientes pasos en tu fuente de datos:

1

Estructurar las columnas en la Hoja de Cálculo

Añade estas columnas exactamente con este tipo de valor en tu archivo de Google Sheets:

Madre_Cabeza_Familia: TEXTO (Valores: "Sí", "No", "No Aplica")
Estado_Civil: TEXTO (Valores: "Soltero(a)", "Casado(a)", "Unión Libre", "Divorciado(a)", "Viudo(a)")
Rango_Edad_Hijos: TEXTO (Valores: "Primera Infancia (0-5)", "Infancia (6-12)", "Adolescencia (13-17)", "Adultos (18+)", "Sin Hijos")
Tiene_Fuero_Sindical: TEXTO (Valores: "Sí", "No")
Tipo_Discapacidad: TEXTO (Valores: "Ninguna", "Física", "Visual", "Auditiva", "Psicosocial")
Hobbie_Principal: TEXTO (Valores: "Deportes", "Lectura", "Cine/TV", "Música", "Viajes", "Manualidades")
2

Cálculo dinámico de Madre Cabeza de Familia

Si solo tienes el dato para las mujeres pero quieres evitar que califique a hombres, utiliza esta fórmula para crear un campo calculado en tu informe:

CASE
  WHEN Genero = "M" THEN "No Aplica"
  WHEN Genero = "F" AND Madre_Cabeza = "Sí" THEN "Sí"
  ELSE "No"
END
3

Métrica de Fuero Sindical y Discapacidad

Crea una Tarjeta de Resultados para el Fuero Sindical y otra para el total de Servidores con alguna discapacidad, aplicando este filtro:

  • Filtro Fuero: Incluir Tiene_Fuero_Sindical = "Sí"
  • Filtro Discapacidad: Excluir Tipo_Discapacidad = "Ninguna"
Consejo de Diseño BI Utiliza la paleta de colores corporativos consistentes (por ejemplo: azul para variables generales, rosa para analítica de familia y morado para seguridad/salud laboral) para facilitar la lectura del reporte institucional.
